# V-SmallestWidthDemo
  AndroidUI适配——smallestWidth适配，或者叫sw限定符适配。
  指的是Android会识别屏幕可用高度和宽度的最小尺寸的dp值（其实就是手机的宽度值），然后根据识别到的结果去资源文件中寻找对应限定符的文件夹下的资源文件。
